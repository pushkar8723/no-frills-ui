# No Frills UI - Storybook Documentation

This branch contains the versioned Storybook documentation for No Frills UI.

## Structure

Each release version has its own directory:
- `/v0.0.13/` - Version 0.0.13 documentation
- `/v0.0.12/` - Version 0.0.12 documentation
- etc.

The `index.html` file at the root provides a landing page that lists all available versions.

## Viewing Documentation

Visit: https://pushkar8723.github.io/no-frills-ui/

## Automated Deployment

Documentation is automatically built and deployed when:
1. A new release is published on GitHub (non-pre-release)
2. Manually triggered via GitHub Actions workflow
