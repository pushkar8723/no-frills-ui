# Radio

Wraps `HTMLInputElement` of type `radio` in a React component to give additional `label` prop.

<Canvas of={RadioStories.Variants} />

```jsx
import { Radio } from 'no-frills-ui';

// Render
<Radio />;
```

## Props

Following props are added on top of `HTMLInputElement` props.

<Controls of={RadioStories.Primary} />

## CSS Variables Used

| Variable Name            | Description                                              |
| :----------------------- | :------------------------------------------------------- |
| `--nfui-primary`         | Used for `color`, `border-color` and `background-color`. |
| `--nfui-primary-light`   | Used for `box-shadow`.                                   |
| `--nfui-background`      | Used for `background` and `box-shadow`.                  |
| `--nfui-border-color`    | Used for `border-color`.                                 |
| `--nfui-disabled-border` | Used for disabled `border-color`.                        |
| `--nfui-light-grey`      | Used for disabled `background-color`.                    |
