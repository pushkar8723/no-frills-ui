# Radio

Wraps `HTMLInputElement` of type `radio` in a React component to give additional `label` prop.

<Canvas of={RadioStories.Variants} />

```jsx

// Render
<Radio />;
```

## Accessibility

The Radio component is fully compliant with WCAG 2.1 AA guidelines:

- **Label Association**: The label is implicitly associated with the radio input through nesting within the `<label>` element.
- **Semantic HTML**: Uses native `<input type="radio">` which provides built-in ARIA semantics and behavior.
- **Keyboard Navigation**: Full keyboard support through native radio button behavior (arrow keys for navigation within a group, Space to select).
- **Focus Indication**: Clear focus styles are provided for keyboard navigation using the `:focus` pseudo-class.
- **Disabled State**: Properly handles disabled state with appropriate visual styling and prevents interaction.
