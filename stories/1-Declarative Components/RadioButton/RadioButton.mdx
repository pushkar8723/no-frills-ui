# RadioButton

Wraps `HTMLInputElement` of type `radio` in a React component. There is also another component `RadioGroup` which
can be used to wrap `RadioButton` component to give a rounded border and a better experience on focus.

`RadioGroup` component is purely for esthetical purpose.

**Example with `RadioGroup`**

<Canvas of={RadioButtonStories.WithRadioGroup} />

**Example without `RadioGroup`**

<Canvas of={RadioButtonStories.WithoutRadioGroup} />

```jsx
import { RadioGroup, RadioButton } from 'no-frills-ui';

// Render
<RadioGroup>
    <RadioButton name="type" label="D" />
    <RadioButton name="type" label="W" />
    <RadioButton name="type" label="M" />
    <RadioButton name="type" label="Y" />
</RadioGroup>;
```

## CSS Variables Used

| Variable Name            | Description                                              |
| :----------------------- | :------------------------------------------------------- |
| `--nfui-primary`         | Used for `color`, `border-color` and `background-color`. |
| `--nfui-primary-light`   | Used for `box-shadow`.                                   |
| `--nfui-background`      | Used for `box-shadow` (inset) and `background-color`.    |
| `--nfui-border-color`    | Used for `border-color`.                                 |
| `--nfui-disabled-border` | Used for disabled `border-color` and `background-color`. |
| `--nfui-light-grey`      | Used for disabled `background-color`.                    |
