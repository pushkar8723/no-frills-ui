# Toggle

Wraps `HTMLInputElement` of type `checkbox` in a React component to give additional `label` prop.

<Canvas of={ToggleStories.Variants} />

```jsx
import { Toggle } from 'no-frills-ui';

// Render
<Toggle />
```

### Props

Following props are added on top of `HTMLInputElement` props.

<Controls of={ToggleStories.Primary} />

### CSS Variables Used

| Variable Name | Description |
| :- | :- |
| --primary | Used for `color` and `border-color`. |
| --primary-light | Used for `box-shadow` and `background-color`. |
