## Accessibility

The Toggle component is fully compliant with WCAG 2.1 AA guidelines:

- **Label Association**: The label is implicitly associated with the toggle input through nesting within the `<label>` element.
- **Semantic HTML**: Uses native `<input type="checkbox">` with `role="switch"` to properly identify this as a switch control for assistive technologies.
- **State Announcement**: The `aria-checked` attribute ensures screen readers announce the current state (on/off) of the toggle.
- **Keyboard Navigation**: Full keyboard support through native checkbox behavior (Space to toggle, Tab to navigate).
- **Focus Indication**: Clear focus styles are provided for keyboard navigation using the `:focus` pseudo-class with visual ring indicator.
- **Disabled State**: Properly handles disabled state with appropriate visual styling and prevents interaction.
