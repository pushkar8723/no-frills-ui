# Chip

Chip component is a `HTMLDivElement` with a remove button. It is intended to be used for inputs where user can enter
multiple values, e.g. input for Languages, Skills, and Hobbies etc.

<Canvas of={ChipStories.Variants} />

```jsx
import { Chip } from 'no-frills-ui';

// Render
<Chip label='Chip 1' />
```
