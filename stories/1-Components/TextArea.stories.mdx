# TextArea

Wraps `HTMLTextAreaElement` in a React component to give additional `label` and `errorText` props.

A `*` is added to the label if the field is required.

<Preview>
    <TextArea
        label='Description'
        minLength={100}
        placeholder='Write anything here...'
        required
    />
</Preview>

```jsx
import { TextArea } from 'no-frills-ui';

// Render
<TextArea />
```
### Props
Following props are added on top of `HTMLTextAreaElement` props.
<Props of={TextArea} />

### CSS Variables Used

| Variable Name | Description |
| :- | :- |
| --primary | Used for `color` and `border-color`. |
| --primary-light | Used for `box-shadow`. |
| --error | Used for `color` and `border-color`. |
| --error-light | Used for `box-shadow`. |
| --background | Used for `background-color`. |
| --border-color | Used for `border-color`. |
