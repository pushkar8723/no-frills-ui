# Radio

Wraps `HTMLInputElement` of type `radio` in a React component to give additional `label` prop.

<Preview>
    <Radio label='Mobile' name='platform' />
    <Radio label='Electron' name='platform' />
    <Radio label='Desktop' name='platform' disabled />
</Preview>

```jsx
import { Radio } from 'no-frills-ui';

// Render
<Radio />
```
### Props
Following props are added on top of `HTMLInputElement` props.
<Props of={Radio} />

### CSS Variables Used

| Variable Name | Description |
| :- | :- |
| --primary | Used for `color` and `border-color` and `background-color`. |
| --primary-light | Used for `box-shadow`. |
| --background | Used for `background` and `box-shadow`. |
| --border-color | Used for `border-color`. |
