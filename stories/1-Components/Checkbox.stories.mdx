# Checkbox

Wraps `HTMLInputElement` of type `checkbox` in a React component to give additional `label`
and `indeterminate` props.

<Preview>
    <Checkbox label='Android' />
    <Checkbox label='iOS' />
    <Checkbox label='Desktop' indeterminate checked disabled />
</Preview>

```jsx
import { Checkbox } from 'no-frills-ui';

// Render
<Checkbox />
```
### Props
Following props are added on top of `HTMLInputElement` props.
<Props of={Checkbox} />

### CSS Variables Used

| Variable Name | Description |
| :- | :- |
| --primary | User for `color` and `border-color`. |
| --primary-light | User for `box-shadow`. |
| --background | User for `background-color`. |
| --border-color | User for `border-color`. |
