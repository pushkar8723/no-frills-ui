# Card

A styled `div`. This component doesn't do much, but helps maintain consistent expirence across other components.

> **Note**: `CardHeader`, `CardBody` and `CardFooter` components are also exported to help with keeping the styles
> consistent with other components and are completely optional to use.

### Props

Props of native`div` element.

| Name | Description | Default |
| :- | :- | :- |
| elevated | Elevates the card. | `true` |

### CSS Variables Used

| Variable Name | Description |
| :- | :- |
| --background | Used for `background-color`. |
| --shadow | Used for `box-shadow`. |
| --modal-shadow | Used for `box-shadow`. |
